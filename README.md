# 🚗 Auto Price Prediction

A Machine Learning project to predict automobile prices using regression techniques.

## 📌 Overview
Built a predictive model to estimate car prices based on 26 features like engine type, horsepower, fuel type, and brand. Achieved **R² score of 0.92** using Random Forest Regression.

## 🛠️ Tools & Technologies
- Python, Pandas, NumPy
- Scikit-Learn (Linear Regression, Random Forest)
- Matplotlib, Seaborn
- Feature Engineering, Label Encoding

## 📊 Results
| Model | R² Score | MAE |
|-------|----------|-----|
| Linear Regression | 0.78 | ~2100 |
| Random Forest | **0.92** | ~1200 |

- Model accuracy improved by **18%** through feature engineering and outlier treatment

## 📁 Dataset
UCI Automobile Dataset — 205 rows, 26 features  
Features include: engine size, horsepower, fuel type, brand, body style, drive wheels, price

## 🔍 Project Steps
1. Data Cleaning & Missing Value Treatment
2. Exploratory Data Analysis (EDA)
3. Feature Engineering & Label Encoding
4. Model Training (Linear Regression + Random Forest)
5. Feature Importance Analysis
6. Actual vs Predicted Visualization
